# Changelog

## 2026-01-27
- 登录/注册页加入语言切换入口（中英）
- 新增“忘记密码”页面与路由
- 登录页增加“忘记密码”入口，并与“去注册”同一行展示
- 登录/注册/忘记密码相关文案补充 i18n（中英文）
- 文件列表支持单击行选择，文件名悬停高亮
- 图片预览拖动更跟手，减少拖拽延迟
- 配置加载支持多路径 .env，并在镜像内生成 /app/config/.env

## 2026-01-28
- 网盘/后台侧栏折叠体验优化（对齐、间距与移动端展开）
- 后台菜单支持一级分组折叠
- 网盘支持批量删除与批量移动（统一列表入参）
- 删除/重命名接口返回结构调整为批量结果
- 重复文件夹/文件创建的错误提示支持 i18n 映射
- Uvicorn 日志输出与路径调整（/app/logs）

## 2026-01-29
- 分享管理检测目标资源是否存在并在列表/公共入口展示“文件已被删除”的状态提醒
- 移动端点击导航时自动收起侧栏，文件面包屑高亮且可交互
- 文件表格与文件工具栏在移动端自适应换行，名称栏底部对齐避免纵向偏移
- 上传接口使用独立超时时间，可通过 `VITE_UPLOAD_TIMEOUT` 控制（默认 120000ms）
